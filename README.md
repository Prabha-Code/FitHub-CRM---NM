# 🏋️ FitnessHub CRM

### Naan Mudhalvan Project

A **complete Gym Management System (CRM)** built using **HTML, CSS, and JavaScript**.
This project helps gyms manage members, trainers, schedules, payments, and analytics — all in one place.

---

## 🚀 Project Overview

**FitnessHub CRM** is a front-end based application designed to streamline gym operations.
It provides an intuitive dashboard and multiple modules to track business performance and user activities.

This project is developed as part of the **Naan Mudhalvan Program (Govt. of Tamil Nadu)**.

---

## ✨ Features

### 📊 Dashboard

* Total members, revenue, renewals
* Monthly growth charts
* Membership distribution
* Real-time activity feed

### 👥 Members Management

* Add / View / Filter members
* Plan & trainer assignment
* Membership status tracking (Active / Trial / Expired)

### 🏅 Trainers

* Trainer profiles
* Ratings & experience
* Member allocation

### 📅 Schedule

* Class calendar view
* Daily class tracking
* Booking status

### 💳 Plans & Pricing

* Multiple membership tiers
* Plan performance analytics

### 💰 Payments

* Track transactions
* Payment methods (UPI / Cash / Card)
* Pending & overdue tracking

### 📈 Reports

* Business insights
* Trainer performance
* Revenue breakdown
* Attendance analytics

### ⚙️ Settings

* Gym details
* Naan Mudhalvan project info form

## ⚙️ How to Run

### 🔹 Option 1: Run Locally

1. Download or clone the repository
2. Open `index.html` in any browser

---

### 🔹 Option 2: Deploy on GitHub Pages

1. Upload project to GitHub
2. Go to **Settings → Pages**
3. Select branch → `main`
4. Get live URL:

```
https://your-username.github.io/fitnesshub-crm
```

---

### 🔹 Option 3: Netlify (1-minute deploy)

1. Go to https://netlify.com/drop
2. Drag & drop project folder
3. Get instant live link
